# IPL Data Analysis Dashboard (Power BI)

This project analyzes Indian Premier League (IPL) data using Microsoft Power BI.

---

## Project Overview
The dashboard provides insights into:
- Team performance
- Top run scorers
- Top wicket takers
- Match trends by season
- Toss decision analysis

---

## Tools Used
- Microsoft Power BI
- DAX
- IPL Dataset (Kaggle)

---

## Dashboard Features
- KPI Cards for total matches, runs, wickets and teams
- Interactive charts and slicers
- Season-based filtering
- Player performance analysis

## Insights
- Mumbai Indians appear among the most successful IPL teams
- Virat Kohli is one of the top run scorers
- Ravichandran Ashwin is among the top wicket takers
- Teams often prefer fielding first after winning the toss
